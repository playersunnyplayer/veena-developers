Enhance EventPress WordPress Themes Functionality.

== Description ==

Evento plugin is provides you a complete theme demo import setup for EventPress WordPress Theme. This Plugin Developed for only EventPress & Childs Theme. EventPress is Seasonal Themes.

The Evento plugin features include:

- Premade Section with Dummy Data
- One Click Data Installation
- 4 predefined content sections
- Live content editing 
- Custom Theme Color
- and many other features
